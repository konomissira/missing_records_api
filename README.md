# Missing Records Detection API

A production-ready REST API for detecting missing records in data pipelines using Python set operations.

## Project Overview

This project demonstrates efficient missing data detection for tracking records through processing pipelines (orders, transactions, files, etc.) and identifying gaps using set difference operations.

## Features

-   Track expected vs processed records
-   Detect missing records using SET DIFFERENCE operations
-   Real-time pipeline monitoring
-   Gap analysis and reporting
-   REST API for pipeline status checks
-   PostgreSQL database for record tracking
-   Docker containerization for easy deployment
-   Comprehensive API documentation

## Tech Stack

-   **Python 3.11**
-   **FastAPI** - Modern web framework
-   **PostgreSQL** - Database
-   **SQLAlchemy** - ORM
-   **Docker & Docker Compose** - Containerization
-   **pytest** - Testing
